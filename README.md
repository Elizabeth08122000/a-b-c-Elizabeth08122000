# (A + B) * C
В консоль поочередно вводятся переменные A, B и C. 
A и B целые числа, а C число с плавающей точкой. 
Вывести результат вычисления по формуле: (A+B)*C

## Формат входных данных
int A  
int B  
float C
## Формат выходных данных
float

## Примеры:
### Входные данные
1  
2  
4.5      
### Результат работы
13.5
